Polymorphism in Python:
=======================
Python supports Polymorphism. Poly means many and morphism means forms in 
Python we can implement using following methodology


->overloading
         ->An operator overloading
         ->Method overloading
         ->Constructor overloading
->overriding
        ->Method overriding
        ->Constructor overriding

=====

import time

    def __init__(self,items):
        self.items=items

import time 
class I_HUB_AA_STORE:
    def __init__(self,items):
        self.items=items 
    def __str__(self):
        return str(self.items)
    def __add__(self,other):
        return self.items+other.items

i1=I_HUB_AA_STORE(265)
print("Number of items in i1 is:",i1)
print()
i2=I_HUB_AA_STORE(195)
print("Number of items in i1 is:",i2)
print()
print("Sum is:",i1+i2)
print()
time.sleep(2)
print("End of an application")
